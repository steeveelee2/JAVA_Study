 - 패키지 추가 : array.silsub2.model
 - 클래스 생성 : array.silsub2.model.Sample.java


[문제 1]
 - 메소드명 : public void silsub1()
 - 구현내용 : 
	 행의 길이가 15, 열의 길이가 11인 2차원 배열을 선언하세요.
	 1부터 165까지 인덱스 순서대로 값을 초기화하고, 
	 그 값을 출력하는 코드를 작성하시오.
	 

[문제 2]
 - 메소드명 : public void silsub2()
 - 구현내용 : 
	 이차원배열의 크기를 1~10까지 입력 받아 랜덤으로 알파벳 소문자 넣기.
	 단, 범위를 벗어나면 “반드시 1~10 사이의 정수를 입력해야 합니다.“ 출력 후 
	 정수를 다시 입력 받도록 작성하시오.
 - hint :
	 2차원배열을 만들어 각 자리에 랜덤값을 넣으면 됨.
	 char형은 숫자를 더해서 문자를 표현할 수 있음.
	 char형 97는 ‘a’이고 1을 더하면 ‘b’가 된다.
	 

[문제 3]
 - 메소드명 : public void silsub3()
 - 구현내용 : 
	 1차원 배열에 12명의 학생들을 출석부 순으로 초기화 하고, 
	 2열3행의 2차원배열 2개를 이용해서 분단으로 나누고, 
	 1분단 왼쪽부터 오른쪽, 1행에서 아래 행으로 순으로 자리를 배치하는 프로그램을 작성하세요.
	 
	 1.홍길동
	 2.이순신
	 3.유관순
	 4.윤봉길
	 5.장영실
	 6.임꺽정
	 7.장보고
	 8.이태백
	 9.김정희
	 10.대조영
	 11.김유신
	 12.이사부

	- 출력 예시 :
	== 1분단 ==
	홍길동  이순신  
	유관순  윤봉길  
	장영실  임꺽정  
	== 2분단 ==
	임꺽정  장보고  
	이태백  김정희  
	대조영  김유신  	 


[문제 4]
 - 메소드명 : public void silsub4()
 - 구현내용 : 
	 3번 문제 자리배치 후 학생이름을 입력 받고, 
	 몇 분단 몇 번째 중 오른쪽/왼쪽 자리인지 검색할 수 있는 프로그램을 작성하세요.
 - 출력 예시
	== 1분단 ==
	홍길동  이순신  
	유관순  윤봉길  
	장영실  임꺽정  
	== 2분단 ==
	임꺽정  장보고  
	이태백  김정희  
	대조영  김유신  
	============================
	검색할 학생 이름을 입력하세요 : 홍길동
	검색하신 홍길동 학생은 1분단 첫 번째 줄 왼쪽에 있습니다.
	검색할 학생 이름을 입력하세요 : 고길동
	검색하신 고길동 학생은 존재하지 않습니다.
